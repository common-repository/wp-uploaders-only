This plugin fixes the bug of allowing users to access and delete other users uploaded content.

== Description ==

Restricts users to access only self uploaded content on the front end. This plugin is coded to works best with plugins like BuddyUpload, etc. It fixes the bug of allowing users to access and delete other users uploaded content.

Instruction: Just install it and leave it be, there's no setting needed.

== Installation ==

1. Upload the plugin zip file to the `/wp-content/plugins/` directory, or install it through the WordPress plugins directory directly.
2. Activate the WP Uploaders Only through the 'Plugins' screen in WordPress and that's it.
